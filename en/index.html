数据管理

易雾微讯

公众号资讯呈现，主要汇集结构化、有深度的二手资讯，借助平台推荐能力，可自动挖掘精品内容。

Dataease

开源数据可视化分析工具，主要用于家庭生产业务数据、运维数据可视化，现有开源模版可用，利于快速入门。

Ev-search

业务数据全文检索，目前主要接入对公众号提供搜索能力。

Gitlab

项目管理和代码托管平台，主要用于优秀开源项目本地留存及自写代码管理。

Grafana

用于可视化大型测量数据的开源程序，提供了强大和优雅的方式去创建、共享、浏览数据。

Hawkeye

GitHub 泄露监控系统，主要用于关注领域的优秀项目发掘。

Crawlab

分布式爬虫管理平台，支持爬虫自动化部署、日志监控、Git 集成、在线文件编辑、消息通知，等等

ELK

日志分析系统，主要用于日志和业务数据的存储、检索及分析。


文件管理

Nextcloud

Nextcloud 网盘存储，主要用于手机重要文件实时同步配合 Nextcloud 手机客户端，Joplin 笔记、Keeweb 多端同步。

可道云

Kodbox 资源管理器，主要用于主力文件收纳，在线办公。

Onlyoffice

在线办公软件，主要给 Kodbox 及 Nextcloud 网盘提供文本文档、电子表格和演示文稿。

Adrive

阿里云盘 Webdav 服务，用于配合 duplicati 、rclone 异地增量备份。


安全管理

内网区 Jumpserver

开源堡垒机，是符合 4A 规范的专业运维安全审计系统，主要用于家庭网络资产统一管理。

办公区 Jumpserver

开源堡垒机，是符合 4A 规范的专业运维安全审计系统，主要用于办公环境下本地资产统一管理。

Ivre

家庭网络资产测绘，IVRE 是一个 Python 网络侦查框架，基于 NMAP，使用 Bro & p0f 和 MongoDB 后端。

Openvas

开放式漏洞评估系统，是 Nessus 项目分支，用于管理目标系统的漏洞，检测目标网络或主机的安全性。

Wazuh

开源安全平台，是一个用于威胁预防、检测和响应的免费开源平台。

Self service password

自助修改域账号密码系统，主要对接 Openldap ，实现用户自助重置或修改 Openldap 密码。

phpLDAPadmin

管理轻量级目录访问协议(LDAP)服务器一个 web 应用程序。

SELKS

网络安全管理系统，基于自己的图形规则管理器提供一套完整的、易于使用的 Suricata 入侵检测及防范生态系统。

Vulfocus

漏洞集成平台，将漏洞环境 docker 镜像，放入即可使用，开箱即用。


通讯影音

Jellyfin

软件媒体系统，用于家庭影音控制和管理媒体和流媒体。

Rocket Chat

开源团队聊天工具，主要功能有群组聊天、私聊群、媒体嵌入、链接预览、文件上传、语音视频聊天，等等。


网关路由

Vnet

出口网关，核心路由控制台。

Vnet-k3

家庭 WiFi 网关控制台。

爱快 iKuai

家庭访客 WiFi 网关控制台。

Pve-lab

Pve-work
